# ✍️ Text to Handwriting (Premium Edition)

A powerful, beautifully designed tool maintained by **Rahul Sharma** that magically converts your typed text into realistic handwriting images and PDFs. 

![Project License](https://img.shields.io/badge/License-MIT-blue.svg)
![Build Tool](https://img.shields.io/badge/Build-Vite-green.svg)

---

## ✨ Features

- **Premium Glassmorphic UI**: Experience a sleek, modern interface with a flawless Dark/Light mode toggle.
- **Rich Text Editor Integration**: Unlike basic converters, highlight any text on your page to reveal a floating toolbar. You can selectively change ink colors, insert bulleted lists, and format text as bold headings.
- **Advanced Paper Customization**: 
  - Choose between multiple paper sizes (A4, A3, A5, Letter, Legal).
  - Define custom line spacing and line color.
  - Apply custom effects like shadows or scanner distortion to make it look like a genuinely scanned photo.
- **Table Drawing**: Easily insert HTML tables directly into your handwritten notes.
- **Freehand Canvas**: Need to add a diagram? Use the integrated 'Draw' canvas feature to sketch directly onto your document.
- **Bring Your Own Handwriting**: Upload your own `.ttf` or `.otf` custom font files to ensure your outputs match your unique handwriting style perfectly.
- **One-Click Export**: Download pages individually as High-Res images or compile them all seamlessly into a single PDF.

---

## 🛠️ Technologies Used

- **Vanilla JS / Vite:** Lightning-fast module bundling and local dev server.
- **html2canvas:** For precise, reliable DOM-to-image capture.
- **jsPDF:** To seamlessly compile your images into downloadable PDF files.
- **Premium CSS Framework:** A ground-up responsive, glassmorphic layout utilizing modern CSS variables and UI structures.

## 💻 Local Setup & Installation

Get the project running locally on your machine in just a few steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/King143rahul/text-to-handwriting.git
   cd text-to-handwriting
   ```

2. **Install the dependencies:**
   ```bash
   npm install
   ```

3. **Start the local development server:**
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173/` (or the URL provided in your console output).

---

## 💡 Pro Tips for Usage

- **Revealing the Toolbar:** Simply select (highlight) a word or sentence on the notebook page to trigger the floating Rich Text Toolbar.
- **Drawing Diagrams:** Click the "Draw (Beta)" button on the sidebar. A sleek pop-up canvas will appear where you can sketch freehand. Click "Add to Paper" when done!
